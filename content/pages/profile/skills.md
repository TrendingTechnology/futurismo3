- HTML
- CSS
- JavaScript
- React

私とプログラミングの出会いでした。それから、HTML, CSSを学び、自分のホームページを作成しました。

また、TechAcademyというオンラインスクールに通い、JavaScript, JQueryを学びました。
修了制作として、Reactフレームワークを利用したフロントエンド開発も学びました。
